Random Flash tips

  • your minimum stage size should be 215 x 138 pixels. this is the size used by many of flash's dialog boxes. like camera requests and storage requests. if its smaller, the docs say the request will fail.
  • try and make your stage match a popular aspect ratio, like 4:3 and 16:9. you can do the math like this: decide your height (ie 720) and do 4 / 3 X 720 (= 960) so 960 is your width and 720 is your height the smaller number should always be the height. (16:9 is 16 / 9 x 720 = 1280 )

Common resolutions and their equivalents in other aspect ratios.

resolution 16:9 4:3 5:4 3:2 1:1
1920 x 1080 16 / 9 x 1080 = 1920 4/3x1080 = 1440
1440 x 1080 4 / 3 x 1080 = 1440 4:3

you can find an aspect ratio by going backward. if you know your resolution is 300 x 240 and want the ratio, try plugging number in to the math. if you do 16 / 9 x 240 you get 426.6 (repeating) so thats not it. if you do 4 / 3 x 240 you get 320 x 240. so thats not right. (answer is 5:4).
